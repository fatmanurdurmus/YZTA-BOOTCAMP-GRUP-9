# Roadmap

## Sprint 1: 19 June 2026 - 5 July 2026

Goal: create the testable foundation.

- Monorepo and governance docs.
- FastAPI backend skeleton.
- React Vite TypeScript frontend skeleton.
- Pydantic strict schemas.
- Deterministic Scope 1, Scope 2, and CBAM-focused Scope 3 calculation engine.
- Law-RAG stub with legal/reference source model.
- LangGraph StateGraph skeleton with loop and timeout guardrails.
- pytest structure.

## Sprint 2: 6 July 2026 - 19 July 2026

Goal: add durable memory and optimization.

- PostgreSQL persistence and migrations.
- pgvector semantic memory for law/factor chunks.
- LangGraph PostgresSaver checkpointing.
- Episodic and procedural memory contracts.
- Green transformation simulation engine.
- Expert feedback loop for factors and formulas.

## Sprint 3: 20 July 2026 - 2 August 2026

Goal: complete demo and reporting.

- Critic Agent depth checks.
- CBAM/SKDM PDF and JSON report generation.
- Mobile-first dashboard.
- End-to-end tests.
- Docker-ready backend and Vercel-ready frontend.
- Final demo video and bootcamp documentation.
